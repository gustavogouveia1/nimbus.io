## O que e Spell Weaving?

**Spell Weaving** (Tecido de Feiticos) e uma arte magica avancada que permite combinar feiticos basicos em sequencia rapida para criar magias devastadoras. Bruxos experientes conseguem "tecer" estas combinacoes enquanto voam em suas vassouras, criando um skill gap real entre iniciantes e mestres.

### Como Funciona

1. Pressione as teclas de magia (1-5) em **sequencia rapida**
2. Voce tem **1.5 segundos** para completar a sequencia
3. Se a sequencia corresponder a um combo, o feitico especial sera lancado
4. Combos consomem **mais mana** mas causam **muito mais dano**

### Controles

| Tecla | Magia Base |
|-------|------------|
| **1** | Stupefy (Atordoamento) |
| **2** | Incendio (Fogo) |
| **3** | Glacius (Gelo) |
| **4** | Bombarda (Explosao) |
| **5** | Protego (Escudo) |
| **TAB** ou **G** | Abrir Grimorio |

## Dicas de Combate

### Para Iniciantes
1. Comece praticando o **Stupefy Triplo** (1-1-1) - e o mais facil
2. O **Fogo Atordoante** (1-2-1) e otimo para combate 1v1
3. Sempre tenha mana suficiente antes de tentar um combo

### Para Intermediarios
1. Use **Paralisia Gelida** (3-1-3) para escapar de perseguicoes
2. **Vapor Explosivo** (2-3-2) e excelente contra grupos
3. Combine combos com o Speed Boost (Shift) para ataques surpresa

### Para Mestres
1. **Harmonia dos Elementos** (1-2-3-4-5) e essencial - dano + cura
2. **Fortaleza Explosiva** para defesa agressiva em grupos
3. **Ciclone Primordial** para controle de area em teamfights
4. Pratique os combos ate conseguir executa-los sem olhar para o teclado
